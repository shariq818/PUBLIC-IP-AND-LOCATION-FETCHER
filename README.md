A simple, interactive Python tool that fetches *public IP information and geolocation data* using the ip-api.com service.  
The user can:
- *Enter any IP address* manually  
- Or press *ENTER to auto-detect their own IP*  
- Results are saved to *ip_info.txt*

Perfect for networking beginners, cybersecurity learners, and portfolio demonstration.
